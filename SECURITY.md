# Security Policy

## Supported Versions

Currently, only the latest version of ProtoSpark is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.0.x  | :white_check_mark: |
| < v0.0.1| :x:                |

## Reporting a Vulnerability

We take the security of ProtoSpark seriously. If you believe you have found a security vulnerability, please report it to us as soon as possible.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to the project maintainers. We will acknowledge receipt of your vulnerability report within 48 hours and send you regular updates about our progress.

After the initial report, we will endeavor to:
1. Validate the vulnerability.
2. Address the issue in a new release.
3. Coordinate a public disclosure.

Thank you for helping keep ProtoSpark secure!
